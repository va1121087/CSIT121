What is XML?

XML (eXtensible Markup Language) is a markup language used for storing and transporting data in a structured format.

How is XML used?

XML is used to represent structured data in a human-readable format. It's commonly used for data exchange between systems and configuration files.

What does XML look like? (Example)

        <person>
            <name>John Doe</name>
            <age>30</age>
        </person>

What is JavaScript?

JavaScript is a versatile programming language commonly used for adding interactivity and functionality to web pages.

How is JavaScript related to HTML and CSS?

JavaScript can be embedded in HTML documents to interact with the content, while CSS is used for styling. Together, they create dynamic web experiences.

Common JavaScript Elements, Properties, Methods, and Functions (Example)

Element: document.getElementById('myElement')

Property: element.textContent

Method: array.push(item)

Function: function calculateSum(a, b) { return a + b; }
